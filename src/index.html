The Wave Nautical Store

Proyecto grupal final. E-commerce de productos náuticos.

◼ Tecnologias: Express, Node.js, JavaScript, React.js, Redux.js, GitHub, JWT, Passport, Google Cloud, MercadoPago, Cloudinary, Formik, LocalStorage, Chart.js, Nodemailer, CSS, Borrado lógico, Dashboard, Railway, Vercel, SweetAlert, PosgretSql ,Sequelize.



×

La página web The Wave es capaz de:



  • Explorar una amplia variedad de productos náuticos como tablas de surf, stand up paddle, wakeboard, aletas, trajes de neopreno, leash y otros accesorios.
  • Filtrar y ordenar los productos por categoría, marca, precio y otros criterios.
  • Ver detalles de los productos, como imágenes, descripciones, especificaciones y disponibilidad.
  • Agregar productos al carrito de compra y ver el total de la compra.
  • Se pueden agregar productos a favoritos y desde ahi mismo agregarlos al carrito.
  • Realizar compras seguras y confiables utilizando diferentes métodos de pago, como tarjetas de crédito y débito a traves de Mercado Pago.
  • Logearse mediante auth0 y google, con las funciones de actualizar los datos del usuario incluyendo un boton para el Logout.
  • Cuenta con dashboard para administrador con borrado lógico, estadísticas, control de stock.
  • Posee review de los productos.

    La página web App Dogs! es capaz de:



  • Explorar razas de perros, con caracteristicas como peso, altura, temperamento, origen,tiempo de vida.
  • Cuenta con barra de busqueda de razas.
  • Filtrar por temperamentos, por origen. Ordenar por nombre y peso. Los filtros se pueden combinar.
  • Cuenta con paginado. Se puede acceder al detalle de raza, volver atras y se mantiene la ubicacion donde nos encontrabamos.
  • Cuenta con una seccion de About.
  • Posee formulario de creacion, donde se puede crear una nueva raza de perro.
  • El formulario cuenta con validacion.
